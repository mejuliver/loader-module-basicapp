Load header assets

		$loader->load('basicapp')->header();
If there are specific plugins you dont want to be loaded then you can set an array of exluded plugins 
		$loader->load('basicapp')->header([
			'bootstrap', // will not be loaded
			'jquery' // will not be loaded
		]);
If you want to use the CDN source, you can forcely tell the plugin to load from cdn source by specifying a true boolean as a 2nd parameter 
		$loader->load('basicapp')->header([],true);
